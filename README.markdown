Plurk-bot
=========

Features
--------
	- A plurk bot which posts the weather forecast on your wall

How to use
----------
	1. Download two .rb files
	2. Open weather.rb and fill in your ACCOUNT, PASSWORD, and APIKEY
	3. Have fun! Maybe you would like to add this into crontab
